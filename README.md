# Financial Analysis Dashboard | Power BI
## Overview
This project is a Financial Analysis Dashboard developed in Power BI using the sample financial dataset
provided by Microsoft Power BI. The dashboard is designed to analyze key business metrics such as sales,
profit, units sold, product performance, and segment-wise contributions through interactive and visually
appealing reports.

This project demonstrates my ability to transform raw business data into meaningful insights using
Power BI, data modeling, and custom DAX measures.

- Dashboard Interaction <a href="https://github.com/aagaz06/powerbi-finance-dashboard/blob/main/PowerBI-Finance-Dashboard.pbix">View Dashboard </a>

## Project Objective
- Monitor overall business performance.
- Track sales and profit trends over time.
- Analyze sales across different customer segments.
- Identify the most profitable products.
- Measure month-over-month growth using DAX calculations.
- Present insights through an intuitive and professional dashboard.

## Key Features
### KPI Cards
- Total Sales
- Total Profit
- Units Sold

### Sales Analysis
- Monthly Sales Trend Analysis
- Segment-wise Sales Performance

### Profit Analysis
- Product-wise Profitability Analysis
- Average Profit by Segment

### Interactive Reporting
- Dynamic visualizations
- Business-focused KPI tracking
- Professional dark-themed dashboard design

## Dashboard Insights
- Government and Small Business segments contribute the highest sales.
- Paseo is the most profitable product.
- October recorded the highest monthly sales.
- Enterprise segment shows comparatively lower profitability.
- Sales performance demonstrates significant growth during the final quarter.

## Project Development Process
- Imported the Power BI Sample Financial Dataset.
- Explored and validated the data.
- Created DAX measures for Sales, Profit, Units Sold, and MoM Growth.
- Designed KPI cards to track key business metrics.
- Built visualizations for sales trends, segment performance, and product profitability.
- Applied a professional dark-themed dashboard design.
- Used conditional formatting to highlight performance changes.
- Optimized layout, colors, and visual hierarchy for better user experience.
- Validated insights and finalized the dashboard for portfolio presentation.
